## API Endpoints

* following is a list of endpoints

### [POST] band/create

| TYPE | URL       | DATA                  |
| ---- | --------- | --------------------- |
| POST | /band/add | bandName*, bandGenre* |

### [GET] /bands

| TYPE | URL       | DATA                  |
| ---- | --------- | --------------------- |
| GET | /bands |  |
| GET | /band/:id | bandId* |

### [DELETE] /band/:id

| TYPE | URL       | DATA                  |
| ---- | --------- | --------------------- |
| DELETE | /api/band/:id | bandId* |

### [EDIT] /band/:id

| TYPE | URL       | DATA                  |
| ---- | --------- | --------------------- |
| EDIT | /band/:id | bandId*, bandName*, bandGenre* |
